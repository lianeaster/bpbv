# Сайт кафедри БПБВ — НУХТ

Офіційний сайт кафедри біотехнології продуктів бродіння і виноробства Національного університету харчових технологій.

**Версія: 1.0**

---

## Що це

Односторінковий сайт з секціями: про кафедру, історія, освітні програми, наука, викладачі, новини, випускники, міжнародна діяльність, гуртки, життя кафедри, галерея, контакти. Підтримка мов: українська, англійська, німецька, французька.

---

## Як запустити локально

```bash
npm start
```

Відкрити в браузері: **http://localhost:3000**

---

## Підтримка та оновлення контенту

**Інструкція для тих, хто підтримує сайт і мало працював з HTML/JS:**

👉 **[ПІДТРИМКА_САЙТУ.md](./ПІДТРИМКА_САЙТУ.md)** — покрокова документація: як змінювати тексти, додавати новини, міняти фото, що можна і не можна редагувати.

Коротко:
- **Тексти** — у файлі `translations.js` (всі мови).
- **Новини** — у файлі `news.json`.
- **Фото** — у папці `images/` (підпапки gallery, science, home, faculty, clubs тощо).

---

## Структура проєкту

| Файл / папка      | Призначення |
|-------------------|-------------|
| index.html        | Головна сторінка (розмітка) |
| style.css         | Стилі (зовнішній вигляд) |
| script.js         | Логіка (меню, переклади, новини) |
| translations.js   | Усі тексти сайту (UK, EN, DE, FR) |
| news.json         | Новини (заголовок, дата, текст, посилання) |
| images/           | Зображення (gallery, science, home, faculty, clubs, …) |
| assets/           | Презентації (PDF) тощо |
