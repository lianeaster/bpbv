# Сайт кафедри БПБВ — НУХТ

Офіційний сайт кафедри біотехнології продуктів бродіння і виноробства Національного університету харчових технологій.

**Версія: 1.2** · [Що змінювалось](./CHANGELOG.md)

---

## Про сайт

Односторінковий сайт кафедри з такими розділами:

| Розділ | Опис |
|--------|------|
| Про кафедру | Загальна інформація, цифри, переваги |
| Історія | Хронологія від 1930 року до сьогодення з детальним описом кожного періоду |
| Освітні програми | Бакалавр, магістр, доктор філософії |
| Наукова діяльність | Напрямки досліджень, публікації, аспіранти, наукові гуртки |
| Викладачі | Профілі викладацького складу |
| Новини | Стрічка новин кафедри |
| Студенти і випускники | Відомі випускники |
| Міжнародна діяльність | Партнерства, обміни, міжнародні проєкти |
| Життя кафедри | Фотохроніка подій: екскурсії, лекції, дегустації |
| Галерея | Фотогалерея кафедри |
| Контакти | Адреса, телефон, пошта, форма зв'язку |

Сайт доступний чотирма мовами: **українська**, **англійська**, **німецька**, **французька**.

---

## Як запустити локально

```bash
npm start
```

Відкрити в браузері: **http://localhost:3000**

Для зупинки — натиснути `Ctrl+C` у терміналі.

## Структура проєкту

```
bpbv/
├── index.html          — розмітка сторінки
├── style.css           — стилі (зовнішній вигляд)
├── script.js           — логіка (меню, переклади, анімації, новини)
├── translations.js     — усі тексти сайту (UA, EN, DE, FR)
├── news.json           — новини (заголовок, дата, текст, посилання)
├── package.json        — налаштування npm (запуск сервера)
├── images/
│   ├── alumni/         — фото випускників
│   ├── clubs/          — фото наукових гуртків
│   ├── faculty/        — фото викладачів
│   ├── gallery/        — фотогалерея
│   ├── home/           — фото головного екрана та секцій
│   ├── international/  — фото міжнародної діяльності
│   ├── logo/           — логотипи кафедри
│   ├── phd/            — фото аспірантів
│   ├── programs/       — фото освітніх програм
│   ├── qr/             — QR-коди
│   ├── science/        — фото наукової діяльності
│   └── student-life/   — фото подій життя кафедри
├── assets/
│   └── presentations/  — PDF-презентації кафедри (UK, EN)
├── CHANGELOG.md        — журнал змін
├── ПІДТРИМКА_САЙТУ.md  — інструкція з оновлення контенту
└── LICENSE
```

---

## Підтримка та оновлення контенту

Для тих, хто підтримує сайт і мало працював з кодом — є окрема покрокова інструкція:

**[ПІДТРИМКА_САЙТУ.md](./ПІДТРИМКА_САЙТУ.md)**

Коротко:
- **Тексти** — файл `translations.js` (усі 4 мови)
- **Новини** — файл `news.json`
- **Фото** — папка `images/` (відповідні підпапки)
- **Презентації** — папка `assets/presentations/`
