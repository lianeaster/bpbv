# Сайт кафедри БПБВ — НУХТ

Офіційний сайт кафедри біотехнології продуктів бродіння і виноробства Національного університету харчових технологій.

**Версія: 1.1**

---

## Що нового (v1.1 — 27 лютого 2026)

- **Історія кафедри** — тепер кожен період можна розгорнути і прочитати детальну інформацію (хто очолював, які вчені працювали, ключові події). Текст взято зі старого сайту кафедри.
- **Секція «Наукові гуртки»** — перенесена всередину секції «Наукова діяльність», щоб все наукове було в одному місці.
- **Меню навігації** — перейменовано пункти: «Міжнародна» → «Міжнародна діяльність», «Життя» → «Життя кафедри». Додано пункт «Наука». Порядок меню відповідає порядку секцій на сторінці.
- **Логотип** — абревіатура перекладається при зміні мови (БПБВ|НУХТ → BPBV|NUFT).
- **Переклад «Викладачі»** — виправлено з «Faculty» на «Teaching Staff» (EN), «Lehrpersonal» (DE), «Enseignants» (FR).
- **Шрифт заголовка** — на іноземних мовах заголовок у hero-секції тепер використовує елегантний шрифт Fraunces з чіткою різницею між великими і малими літерами.
- **Аспіранти** — три картки аспірантів тепер показуються в один рядок на комп'ютері.
- **Бульбашки на мобільних** — виправлена анімація, яка раніше починалася з середини екрану.
- **Фото «Життя кафедри»** — виправлено відповідність фотографій до подій.

---

## Що це

Односторінковий сайт з секціями: про кафедру, історія, освітні програми, наука, викладачі, новини, випускники, міжнародна діяльність, життя кафедри, галерея, контакти. Підтримка мов: українська, англійська, німецька, французька.

## Підтримка та оновлення контенту

**Інструкція для тих, хто підтримує сайт і мало працював з HTML/JS:**

👉 **[ПІДТРИМКА_САЙТУ.md](./ПІДТРИМКА_САЙТУ.md)** — покрокова документація: як змінювати тексти, додавати новини, міняти фото, що можна і не можна редагувати.

Коротко:
- **Тексти** — у файлі `translations.js` (всі мови).
- **Новини** — у файлі `news.json`.
- **Фото** — у папці `images/` (підпапки gallery, science, home, faculty, clubs тощо).

---

## Структура проєкту

| Файл / папка      | Призначення |
|-------------------|-------------|
| index.html        | Головна сторінка (розмітка) |
| style.css         | Стилі (зовнішній вигляд) |
| script.js         | Логіка (меню, переклади, новини) |
| translations.js   | Усі тексти сайту (UK, EN, DE, FR) |
| news.json         | Новини (заголовок, дата, текст, посилання) |
| images/           | Зображення (gallery, science, home, faculty, clubs, …) |
| assets/           | Презентації (PDF) тощо |
